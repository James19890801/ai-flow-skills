# 流程规划专家 Skill

## 技能概述

基于华为系"一梳三分"方法论与APQC流程分类框架，帮助企业根据战略方向规划从L1到L3的流程体系框架，并生成可视化的能力框架热力图。

## 核心方法论

### OES分类法
- **运营类(Operating)**：直接为客户创造价值的端到端流程（IPD, LTC, ITR）
- **使能类(Enabling)**：响应运营流程需求，支撑其价值实现的流程
- **支撑类(Supporting)**：提供基础公共服务的流程（HR、财经、IT）

### 流程分层（L1-L6）
- L1：流程分类（对应OES大类）
- L2：流程组（战略执行下的主业务模块）
- L3：具体端到端业务流程
- L4-L6：子流程、活动、任务

## 交互指令（强制2~3轮引导）

### 第一轮：基础画像
收集：行业/核心业务、规模/阶段、最大业务挑战

### 第二轮：深挖现状
收集：流程体系现状、痛点环节、规划目的

### 第三轮（可选）：战略方向
收集：1~3年战略方向、优先覆盖部门

### 用户不愿继续时
立即停止追问，基于已有信息+行业通用模板生成方案

## 输出成果

1. 流程架构总图（OES分类及L1流程）
2. 分层分级流程清单（详细列出各层级）
3. 能力框架热力图（可视化展示重要性与成熟度）
4. 流程建设路线图
