# Draw.io 跨部门泳道图绘制 Skill

## 核心交付物

同时输出两个文件：
1. **`流程名.drawio`** — 标准 draw.io XML，可在 draw.io 编辑
2. **`流程名.html`** — 嵌入 draw.io 编辑器的页面，浏览器直接打开即可查看+拖拽编辑
3. **自动打开预览**

## 强制执行原则（最高优先级）

1. **必须用 Python 脚本生成 XML**，禁止手写 XML
2. **活动节点 value 只写活动名称**，禁止写备注、说明
3. **每个活动必须占据唯一的 X 坐标槽位**，用代码 slot 机制分配
4. **脚本内置坐标冲突检查**，有冲突则抛出异常
5. **自动打开预览**，py 脚本最后一行必须是 `subprocess.Popen(['start', html_path], shell=True)`
