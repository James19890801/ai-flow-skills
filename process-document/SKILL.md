# 流程文件生成专家 Skill

## 核心交付物

生成包含13个标准模块的完整流程文件：

1. 流程名称（命名规范：范围+业务对象+动词+类型）
2. 目的
3. 适用范围
4. 术语定义
5. 输入/输出
6. 关键活动（KCP关键控制点）
7. 角色/职责（RACI矩阵）
8. 流程图（Mermaid语法）
9. 活动说明（5W2H，每条≥50字）
10. 相关流程文件
11. 相关流程
12. 相关标准
13. 流程绩效指标（KPI表格）
14. AI提效建议

## 执行流程

```
用户描述流程需求
    ↓
【第1轮】信息采集（必须至少2轮追问）
    - 流程类型与目的
    - 核心KPI
    ↓
【第2轮】流程边界确认
    - 起点：触发事件
    - 终点：完成标志
    - 中间：主要环节
    ↓
生成完整13模块流程文件
```
